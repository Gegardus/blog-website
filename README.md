# Blog-Website

>  The Blog app will let users view post blogs, make comments and likes.

## Built With

- Major languages (Ruby)
- Framworks (Ruby on Rails)

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- A text editor(preferably Visual Studio Code)

### Install

- Ruby
- Ruby on Rails
- PostgresSQL
- Rspec

### Using it Locally

- Clone the project

### Setup

Install gems with:

```
bundle install
```

Setup database with:

```
rails db:create
rails db:migrate
rails db:seed
```

### Usage

Start server with:

```
rails server
```

Open http://localhost:3000/ in your browser.

### Install linter

```bash
bundle install
```

### Run tests

Install rspec with:

```
bundle install
```

and

```
rails generate rspec:install
```
run the test with:

```
rails exec rspec
```
